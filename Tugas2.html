tugas 2 Coffee_Production

Tugas 2 : Prediksi Jumlah Produksi Kopi pada Negara Brazil.

Data Understanding

Dataset produksi kopi ini menyajikan informasi historis tentang produksi kopi di berbagai negara selama periode waktu tertentu, dimulai dari musim panen 1990/91 hingga 2019/20. Data ini terdiri dari variabel-variabel kunci, seperti negara penghasil kopi, jenis kopi yang diproduksi, serta jumlah produksi tahunan.

Dataset yang digunakan dalam pengerjaan kali ini didapat dari kaggle https://www.kaggle.com/datasets/michals22/coffee-dataset lebih tepatnya pada bagian datasets produksi kopi.

Pada tugas kali ini penulis menggunakan prediksi produksi kopi lebih mengerucutkan untuk memprediksi jenis kopi apa yang banyak di produksi pada negara Brazil.

Pada tugas kali ini menggunakan 4 metode yaitu Regresi, DecisionTree, SVR(Support Vector Regression) dan Random Forest. dari 4 metode tersebut

Dari code diatas telah memberikan informasi mengenai beberapa fitur yang ada pada dataset tentang Kopi. Fitur yang di dapat dalam dataset ini adalah 32 fitur dengan fitur total_produksi sebagai target. Selain itu terdapat fitur lain nya yaitu :

  • Country (Negara) : fitur ini berisi nama negara yang memproduksi kopi. Dengan type data kategori (Categorical)
  • Coffee Type (Jenis Kopi) : fitur ini berisi jenis kopi yang diproduksi oleh negara tersebut. Kopi ini diklasifikasikan menjadi 2 jenis kopi yaitu Arabika dan Robusta. Dengan type data kategori (Categorical)
  • 1990/91, 1991/92, 1992/93, dst. (keterangan produksi kopi tahunan) : fitur ini berisi mengenai jumlah produksi kopi id negara tersebut pada periode panen tertentu
  • Total Production (Total Produksi) : fitur ini menunjukkan total produksi kopi dari suatu negara selama periode tertentu. Nilai ini menujukkan jumlah dari seluruh produksi tahunan dari setiap negara.
  • Data Understanding

Pada dataset ini kita memiliki banyak data dari berbagai negara yang memproduksi berbagai macam jenis kopi. Pada tugas kali ini berfokus pada negara Brazil yang jenis kopinya adalah Arabica/Robusta. kali ini tugas ini akan memprediksi jumlah kopi yang akan di produksi pada tahun berikutnya.

Outlier pada dataset Coffee_production.csv

Outlier adalah data yang berbeda secara signifikan dari data lain dalam sebuah set data. Mereka bisa menjadi nilai yang sangat kecil atau sangat besar dibandingkan dengan nilai rata-rata atau mayoritas data lainnya. Pada code ini akan menampilkan outlier terkhusus pada negara brazil karen pada proyek ini terkhusus untuk negara Brazil.

In[9]:
import pandas as pd
    import matplotlib.pyplot as plt
    
    # Membaca dataset
    url = 'https://raw.githubusercontent.com/Auliyaaas/Proyek-Sains-Data/main/PSD/CoffeeProduction/Coffee_production.csv'
    df = pd.read_csv(url)
    
    # Memeriksa kolom yang tersedia dalam dataset
    print(df.columns)
    
    # Pastikan nama kolom negara sesuai, misalnya 'Country'
    # Memfilter data hanya untuk negara Brazil
    brazil_data = df[df['Country'] == 'Brazil']
    
    # Menampilkan beberapa baris pertama data Brazil
    print(brazil_data.head())
    
    # Memilih kolom numerik untuk analisis
    numeric_columns = brazil_data.select_dtypes(include=['float64', 'int64']).columns
    print(f"Kolom numerik: {numeric_columns}")
    
    # Memeriksa outlier menggunakan metode IQR untuk setiap kolom numerik
    for column in numeric_columns:
        Q1 = brazil_data[column].quantile(0.25)  # Kuartil pertama
        Q3 = brazil_data[column].quantile(0.75)  # Kuartil ketiga
        IQR = Q3 - Q1  # Interquartile Range
        lower_bound = Q1 - 1.5 * IQR
        upper_bound = Q3 + 1.5 * IQR
    
        outliers = brazil_data[(brazil_data[column] < lower_bound) | (brazil_data[column] > upper_bound)]
        print(f"\nKolom: {column}")
        print(f"Jumlah outlier di Brazil: {len(outliers)}")
        if not outliers.empty:
            print(outliers[[column]])
    
    # Visualisasi menggunakan boxplot
    plt.figure(figsize=(15, 10))
    brazil_data[numeric_columns].boxplot()
    plt.title("Boxplot untuk Memeriksa Outlier di Brazil")
    plt.show()

Pemodelan

Pemodelan dalam penambangan data adalah alat untuk mengubah data mentah menjadi wawasan berharga yang dapat digunakan untuk mendukung keputusan strategis dalam berbagai bidang. Dalam pengerjaan ini digunakan beberapa pemodelan yaitu sebagai berikut:

Linear Regression

Linear Regression adalah metode regresi sederhana yang memodelkan hubungan linier antara variabel input (X) dan output (Y). Model ini mencoba mencari garis lurus terbaik (best-fit line) yang meminimalkan selisih kuadrat (least squares) antara nilai prediksi dan nilai aktual.

Support Vector Regression (SVR)

Model ini mencoba menemukan garis regresi dalam margin tertentu () sehingga kesalahan prediksi yang kecil tidak diperhatikan. SVR dapat menggunakan kernel seperti linear, polynomial, dan RBF untuk menangani data non-linier.

Decision Tree Regression

Decision Tree membagi data secara rekursif berdasarkan fitur untuk membuat model pohon keputusan. Model ini mempartisi data ke dalam beberapa "daun," di mana setiap daun berisi prediksi nilai rata-rata dari data di dalamnya.

  • Algoritma Dasar:
  1. Memilih fitur terbaik untuk membagi data (berdasarkan pengurangan impurity seperti MSE atau MAE).
  2. Membagi data hingga memenuhi kriteria stopping (misalnya, kedalaman pohon maksimum atau jumlah data minimum di daun).

Random Forest Regression

Random Forest adalah metode ensemble yang menggunakan banyak Decision Trees. Prediksi akhir adalah rata-rata prediksi dari semua pohon di ensemble. Model ini mengurangi masalah overfitting yang sering terjadi pada Decision Tree dengan menggabungkan hasil dari beberapa pohon.

        print(f"RMSE: {rmse:.2f}")
        print(f"MAE: {mae:.2f}")
        print(f"R-squared: {r2:.4f}")
    
        return rmse, model

MAPE (Mean Absolute Percentage Error)

MAPE (Mean Absolute Percentage Error) adalah ukuran kesalahan prediksi yang digunakan untuk mengevaluasi seberapa baik suatu model memprediksi nilai target dibandingkan nilai aktual.

import numpy as np
    
    # Contoh nilai aktual dan prediksi
    y_actual = np.array([3164400000, 1975980000, 3281340000, 2334660000, 2205660000, 2854680000,
     3907860000, 3198300000, 2901120000])
    y_predicted = np.array([3.40728e+09, 2.35686e+09, 3.32508e+09, 2.90592e+09, 1.56888e+09, 1.87860e+09,
     3.49266e+09, 3.17226e+09, 1.88190e+09])
    
    # Fungsi untuk menghitung MAPE
    def mean_absolute_percentage_error(y_actual, y_predicted):
        # Pastikan tidak ada pembagian dengan nol
        non_zero_mask = y_actual != 0
        y_actual = y_actual[non_zero_mask]
        y_predicted = y_predicted[non_zero_mask]
    
        # Rumus MAPE
        mape = np.mean(np.abs((y_actual - y_predicted) / y_actual)) * 100
        return mape
    
    # Menghitung MAPE
    mape = mean_absolute_percentage_error(y_actual, y_predicted)
    print(f"MAPE: {mape:.2f}%")
